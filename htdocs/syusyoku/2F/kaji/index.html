1. 料理

日々の食事に関する一連の作業。

1-1. 計画

  • 献立作成: 定番や挑戦したい料理を10個ほどリストアップしておくと、飽きずに楽しめる。作り置き可・不可を明記するテンプレを用意すると便利。
  • 食材管理: 冷蔵庫・冷凍庫の中身を把握し、賞味期限・消費期限も確認する。先入れ先出し（FIFO）を意識する。

作り置きテンプレ（メモ）

  • 料理名：＿＿＿＿　作り置き：可/不可　保存方法：冷蔵/冷凍　保存期間目安：＿＿＿

1-2. 買い出し

  • まとめ買い: 買い出しの頻度を抑え、冷蔵庫の容量に合わせてまとめ買いをすると無駄遣いが減る。
  • 買い物メモ（順序の小ワザ）: 冷蔵品→冷凍品→常温品→最後にカゴへ。帰宅後は冷たいものを優先で冷蔵/冷凍へ。

1-3. 調理

  • 下準備: 食材を洗う、切るなどの準備。
  • 調理: 焼く、煮る、炒めるなどの加熱調理。

1-4. 保存と食品衛生

  • 冷蔵・冷凍の目安温度: 冷蔵庫は目安で4°C以下、冷凍庫は-18°C以下を推奨（機器差があるので取扱説明書を参照）。
  • 保存方法: 生ものと加熱済みは分ける。密閉容器やラップで保存し、ラベルに「作成日」を書く。
  • 残り物の扱い: 冷蔵は2〜3日以内が目安、冷凍は約1か月目安（料理・材料による）。再加熱は中心温度が十分に上がるまで加熱する。

1-5. 後片付け

  • 食器洗い: 使った調理器具や食器を洗う。
  • 収納: 洗った食器などを拭き、食器棚に片付ける。

2. 洗濯

衣類を清潔に保つための作業。

2-1. 洗濯前の準備

  • ポケットの確認: ポケットの中にティッシュなどが入っていないか確認する。
  • 仕分け:
    • 洗濯表示: 洗濯表示（温度、漂白不可、ドライ、タンブル乾燥の可否）を確認して分ける。おしゃれ着は専用洗剤を使用。
    • 色物: 色移りを防ぐため、濃い色の衣類と白物を分ける。
  • 前処理: 食べこぼし等のシミは、洗濯前にシミ抜きやつけ置き洗いを行う。
  • 洗濯ネットの使用: 型崩れや傷みを防ぎたい衣類はネットに入れる。

2-2. 洗剤・柔軟剤のポイント

  • 洗剤の量: 洗濯機の容量や水量に合わせて適量を使う（目安：液体洗剤のキャップ1杯など。製品の指示を確認）。
  • 入れ方: 洗剤投入口や洗濯槽の指示に従う。過剰使用は衣類や洗濯機の臭いの原因。
  • 柔軟剤: 必要に応じて使用。機能性衣類や吸水性を落としたくないタオルには注意。

2-3. 洗濯の手順

  • 洗濯機を回す（洗濯表示・容量を確認）。
  • 洗濯物を干す: 洗濯後はすぐに干す。放置は臭いの原因になる。
  • 洗濯物を取り込み、畳む。
  • 収納: 畳んだ衣類をクローゼットやタンスにしまう。
  • アイロンがけ: シワが気になる衣類にアイロンをかける。

2-4. 干す際のポイント・生乾き対策

  • 干す前に振る: シワを伸ばし、乾きを早くする。
  • 間隔を空ける: 風通しを良くして、生乾きを防ぐ。
  • アーチ状に干す: 両端に長い衣類、中央に短い衣類を干すと風が通りやすい。
  • 部屋干し対策: 換気を良くする、扇風機や除湿機を併用、部屋干し用消臭剤の利用。
  • 洗濯槽の管理: 定期的に槽洗浄（洗濯槽クリーナー）を行い、カビ・臭いを防ぐ。

3. 掃除

住居を清潔で快適な状態に保つ作業。

3-1. 掃除の基本

  • 片付け: 散らかった物を定位置に戻す。
  • ホコリ取り: 「上から下へ」を意識し、照明器具や棚の上から始める。
  • 床掃除: 掃除機をかけ、必要に応じて拭き掃除をする。
  • ゴミまとめ: 各部屋のゴミ箱のゴミを集めて処分する。

3-2. 換気・カビ対策

  • 換気: 風呂・洗面所は使用後に換気する。窓を開ける、換気扇を回す。
  • カビ取り: 湿気の溜まりやすい箇所は定期的にチェックし、発見次第カビ取り剤で処理。

3-3. 洗剤の選び方（簡易）

  • 油汚れ用（キッチン用）、カビ取り用（浴室）、マルチクリーナー（日常使い）などを使い分ける。
  • 注意: 洗剤同士の混合は禁止。危険（有害ガス発生）になる場合がある。

3-4. 場所別の掃除リスト（頻度目安）

リビング・寝室

  • 【毎日～3日に1回】床掃除（掃除機、フローリングワイパー）
  • 【週に1回】棚やテレビ台のホコリ取り、テレビ画面を拭く、寝具の洗濯
  • 【月に1回】照明器具の掃除、窓拭き、サッシの掃除、エアコンフィルターの掃除（目安）
  • 【季節ごと】ラグの掃除、布団の掃除機かけ・天日干し

キッチン

  • 【毎日】シンクの掃除、コンロ周りの拭き掃除
  • 【週に1回】排水口の掃除、調理家電の外側を拭く
  • 【月に1回】冷蔵庫内の掃除、換気扇の掃除（フィルターなど）

風呂・洗面所

  • 【毎日】浴槽の軽い洗浄、排水口の髪の毛取り
  • 【週に1回】排水口の本格掃除、床・壁の掃除、鏡を拭く、マット類の洗濯
  • 【月に1回】カビ取り、洗濯機の掃除（洗濯槽クリーナーなど）

トイレ・玄関

  • トイレ: 【週に1～2回】便器、床、壁の掃除
  • 玄関: 【週に1回】床の掃き掃除、たたきの水拭き。下駄箱は月1回掃除。

4. 日用品の管理と収納

モノを管理し、暮らしやすく整える作業。

4-1. 在庫管理

  • 在庫チェック: トイレットペーパー、洗剤、ゴミ袋など日用品の在庫を確認する。ストック数の目安を決める（例：トイレットペーパー3ロール、洗剤1本など）。
  • FIFO: 新しいものは後ろへ、古いものを先に使う先入れ先出しを徹底する。

4-2. 収納と補充

  • 補充・交換: 無くなったものを新しいものと交換し、ストックを補充する。
  • 定位置管理: モノの住所を決め、どこに何があるか一目でわかる状態にする。必要に応じてラベリングも活用する。
  • ラベリング例: 薬や消耗品に「開封日」を貼ると管理しやすい。

5. 家計管理

お金の流れを把握し、計画的に管理する作業。

  • 支出の把握: 食費、光熱費、通信費などの項目ごとに支出を記録する。
  • 簡易テンプレ: 月次収入 → 必須支出（家賃等）→ 可変支出 → 貯蓄 の順で管理する。家計アプリやCSVテンプレを用意すると便利。
  • 貯蓄管理: 先取り貯金など、計画的に貯蓄を行う。
  • 重要物の管理: 通帳、印鑑、保険証券などをまとめて保管する。災害時の持ち出し袋にコピーや必要書類を入れる。

6. 住まいの維持管理

生活環境を安全・快適に保つためのメンテナンス作業。

  • 消耗品の交換: 電球やリモコンの電池交換。
  • 定期点検の目安: 煙探知機の電池点検：6か月毎、エアコンフィルタ：1〜3か月目安、水まわりのパッキン確認を定期的に行う。
  • 軽微な修繕: 緩んだネジを締めるなど、簡単な修理。専門が必要な場合は業者へ。
  • 設備の点検: 家電や住宅設備の簡単な動作確認。異常があれば管理会社／大家に連絡する。

緊急対応の初動テンプレ

  • 水漏れ: 元栓の場所を確認して止水、大家または管理会社に連絡。
  • ガス臭い: 元栓を閉める、窓を開ける、火気厳禁、ガス会社へ連絡。緊急の場合は避難と119/ガス会社の緊急連絡先へ。

7. 季節ごとの家事

季節の変わり目に行う作業。

  • 衣替え
  • 寝具の入れ替え（夏用・冬用など）
  • エアコンの本格的な掃除
  • 年末の大掃除
  • 花粉・梅雨対策: 室内干しの工夫、加湿器／除湿機の使い分け、換気を意識する。
  • 防寒準備: 窓のすきまテープや断熱カーテンの使用を検討する。
